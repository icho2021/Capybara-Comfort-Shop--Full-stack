Storefront background (tiled)

1) Add your image under: client/public/images/  (e.g. background.png)

2) Use the same path in two places:
   - api/prisma/storefront-background.path.js  -> STORE_BACKGROUND_URL
   - client/src/index.css -> :root and [data-theme="dark"] --shop-bg-image (wrap in url("..."))

3) Apply to DB:  cd api && npm run seed:storefront

4) Tiling: edit body background-size / background-repeat in index.css (last layer is the pattern).
